## 构建镜像本地调试

修改镜像构建文件，先拉取go语言依赖，利用docker缓存机制，设置加速，设置启动命令。

拉取的镜像从本地开发镜像仓库拉取，加快镜像构建速度。

### 服务端调试

#### 构建服务端镜像

```bash
docker buildx build --file Dockerfile-frps \
--platform linux/amd64 --tag firfe/frps:0.67.0 \
--output type=docker .
```

#### 创建必要文件

```bash
mkdir -p ts-调试/frps && touch ts-调试/frps/frps.toml && touch ts-调试/frps/compose.yaml
```

##### 配置文件 frps.toml

```bash
cat <<EOF > ts-调试/frps/frps.toml
# frps.toml - 服务端配置
# 基础配置
## frps 服务端-客户端 通讯端口
bindPort = 7000

# 安全配置
[auth]
method = "token"
token = "frps"

# Web 管理界面
[webServer]
addr = "0.0.0.0"
## 管理界面端口
port = 7500

EOF
```

##### 部署文件 compose.yaml

```bash
cat <<EOF > ts-调试/frps/compose.yaml
name: frps
services:
  frps:
    container_name: frps
    image: firfe/frps:0.67.0
    network_mode: host
    tty: true
    stdin_open: true
    volumes:
      - /etc/localtime:/etc/localtime:ro
      - /etc/timezone:/etc/timezone:ro
      - ./frps.toml:/etc/frp/frps.toml

EOF
```

#### 运行命令

- 启动
  ```bash
  docker compose -f ts-调试/frps/compose.yaml up -d
  ```
- 启动新构建镜像
  ```bash
  docker compose -f ts-调试/frps/compose.yaml up -d --force-recreate
  ```
- 重启
  ```bash
  docker compose -f ts-调试/frps/compose.yaml restart
  ```
- 删除
  ```bash
  docker compose -f ts-调试/frps/compose.yaml down
  ```

# 网页界面

webServer.pprofEnable = false

#[[proxies]]            # 配置代理快

#type = "tcp"           # 代理类型 无特殊情况填写 tcp
#localIP = "127.0.0.1"  # 本地 IP
#localPort = 2345       # 本地 端口
#remotePort = 1212      # 远程端口，不写动态分配

type = "tcp"

## 构建 Win 版本

- 创建容器
  ```bash
  docker run --rm -it --name KaiFa_exe -h KaiFa_exe --restart=no \
  --cpus=1 --memory=2G --network host \
  -e VERSION=0.67.0 -e CGO_ENABLED=0 -e GOOS=windows -e GOARCH=amd64 \
  -v ./ts-调试/go-mod:/go/pkg/mod -v ./ts-调试/go-cache:/.cache \
  docker.code.firfe.work/golang:1.24 bash
  ```
- 复制文件 容器外执行
  ```bash
  docker cp ./代码 KaiFa_exe:/app
  ```
- 设置环境变量
  ```bash
  go env -w GOPROXY=https://mirrors.aliyun.com/goproxy/,direct && \
  LDFLAGS="-s -w -X github.com/fatedier/frp/pkg/util/version.Version=${VERSION}" && \
  cd /app
  ```
- 编译服务端
  ```bash
  go build \
  -trimpath \
  -ldflags "${LDFLAGS}" \
  -tags frps \
  -o ./release/frps_windows_amd64_${VERSION}.exe \
  ./cmd/frps
  ```
  - 从容器中复制编译后的软件到本地 容器外执行
    ```bash
    docker cp KaiFa_exe:/app/release/frps_windows_amd64_0.67.0.exe ./ts-调试/
    ```
- 编译客户端
  ```bash
  go build \
  -trimpath \
  -ldflags "${LDFLAGS}" \
  -tags frpc \
  -o ./release/frpc_windows_amd64_${VERSION}.exe \
  ./cmd/frpc
  ```
  - 从容器中复制编译后的软件到本地 容器外执行
    ```bash
    docker cp KaiFa_exe:/app/release/frpc_windows_amd64_0.67.0.exe ./ts-调试/
    ```
